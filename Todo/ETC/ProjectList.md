# 프로젝트 아이디어 정리

## 1. 화실🎨
### 🖌️ 핵심 컨셉
- 그린 그림을 온라인 갤러리 형식으로 전시
- 방문자가 그림을 감상하고, 댓글/리뷰를 남길 수 있도록
- 그림에 대한 설명, 제작 과정, 스토리 등을 함께 제공

### 📌 주요 기능
- **그림 전시 페이지**: 갤러리 스타일의 카드 UI
- **그림 카테고리 및 필터**: 장르별(풍경, 인물화 등) / 제작 연도별 분류
- **좋아요 & 댓글 기능**: 로그인한 유저가 댓글 남기고 좋아요 가능
- **관리자 페이지 (엄마 전용 업로드 기능)**: 그림 업로드, 수정, 삭제 가능
- **반응형 디자인**: 모바일에서도 그림 감상 최적화
- **(선택) 그림 구매 기능**: 관심 있는 방문자가 구매 요청 가능

### 📌 개발 기술
- **Frontend**: Vue.js, React
- **Backend**: Spring Boot (Java) / Express.js (Node.js)
- **Database**: MySQL / Firebase
- **Storage**: AWS S3 (그림 저장)

### 🎯 확장 아이디어
- 그림마다 "그리는 과정" 동영상 추가
- 방문자 AI 추천 기능 (선호하는 스타일 분석)
- AI 기반 "내 사진을 그림 스타일로 변환" 기능

## 2. 클라이밍 커뮤니티 & 지도 서비스 🧗‍♂️
### 🏔️ 핵심 컨셉
- 전국 클라이밍장 정보를 모으고, 커뮤니티 기능 제공
- 클라이머들이 루트 정보 공유, 후기 남기기
- 초보자를 위한 튜토리얼 제공

### 📌 주요 기능
- **클라이밍장 지도 & 검색**: 위치 기반 검색 (가까운 클라이밍장 추천), 가격, 난이도, 시설 정보 제공
- **커뮤니티 (게시판)**: 루트 후기, 장비 추천, 운동 팁 공유
- **내 클라이밍 기록 저장**: 오늘 등반한 루트 기록, 사진 업로드
- **레벨 시스템**: 사용자가 자신의 등반 레벨 설정 (V0~V10)
- **(선택) 온라인 매칭 기능**: 같이 등반할 사람 구하기 기능

### 📌 개발 기술
- **Frontend**: Vue.js + Tailwind
- **Backend**: Firebase (빠른 MVP) or Spring Boot
- **Database**: Firestore / PostgreSQL
- **지도 서비스**: Google Maps API / Kakao 지도 API

### 🎯 확장 아이디어
- **AI 등반 분석** (영상 업로드 시 자세 분석)
- **경쟁 요소** (루트 클리어 인증 & 랭킹)
- **전문가 코칭 기능** (클라이밍 트레이너 연결)

## 3. 블로그 (개인 기술 블로그 & 프로젝트 정리) ✍️
### 🖥️ 핵심 컨셉
- 개발 경험, 공부한 내용, 프로젝트 정리하는 블로그
- Markdown 기반 글쓰기 & 깔끔한 UI
- 방문자 통계, 다크 모드 지원

### 📌 주요 기능
- **게시글 작성 & 카테고리 관리**: Markdown 지원 (코드 블록 포함), 태그 기능 (#Java, #Vue)
- **검색 & 필터링 기능**: 프로젝트별, 기술별 분류
- **방문자 통계 대시보드**: 방문자 수, 인기 글 분석
- **(선택) 커스텀 도메인 & SEO 최적화**: 검색 엔진 노출 최적화
- **(선택) 댓글 & 반응 기능**: GitHub 로그인 댓글 시스템

### 📌 개발 기술
- **Frontend**: Vue.js / Nuxt.js
- **Backend**: Firebase / Spring Boot
- **Database**: Firestore / PostgreSQL
- **Hosting**: Vercel / Netlify / AWS

### 🎯 확장 아이디어
- **AI 요약 기능** (긴 글 요약 자동 생성)
- **블로그 포트폴리오 기능** (내 프로젝트 전시)
- **방문자 맞춤 추천 게시글**

## 4. 개발자용 커밋 챌린지 & 기록 웹앱 🚀👨‍💻
### 개요
하루에 한 번 이상 GitHub에 커밋하면 streak을 유지하고, 목표를 달성하면 뱃지를 받을 수 있는 웹앱.

### 📌 주요 기능
- GitHub API 연동 → 자동 커밋 확인
- 리더보드 & 랭킹 시스템
- 개인 목표 설정 (예: 30일 커밋 챌린지)
- 다크 모드 지원

### 기술 스택
- **Frontend**: Vue.js / Next.js
- **Backend**: Node.js (Express)
- **Database**: MariaDB
- **OAuth 로그인**: GitHub

## 5. 🎮📺 개인 감상 기록 & 평점 사이트
### 개요
나의 취미생활을 저장하면서 기록하는 공간.

### 아이디어
- 본 게임 / 애니메이션 / 미드(드라마) / 영화를 기록하고 평점을 남길 수 있는 개인 기록 웹앱
- 감상 날짜, 장르, 감상평, 추천 여부 등을 정리할 수 있음
- 나만의 ‘Top 10’ 같은 기능 추가 가능

### 📌 주요 기능
✅ private, public 선택 확인
✅ 유저들의 평점 랭킹
✅ 개인 목표 설정 (예: 몇월달에 무슨 게임, 무슨 드라마)
✅ 다크 모드 지원
✅ 기록 기능: 본 콘텐츠(게임, 애니, 미드, 영화) 등록 + 감상평 작성
✅ 평점 시스템: 별점(⭐) or 10점 만점 평가 + 추천 여부 체크
✅ 태그 & 검색 기능: 장르별, 국가별, 감독/제작사별 정렬 가능
✅ 통계 대시보드: 가장 많이 본 장르, 평균 평점 등 시각화
✅ To-Watch 리스트: 보고 싶은 목록 따로 저장 가능


### 기술 스택
- **Frontend**: Vue.js / Next.js
- **Backend**: Node.js (Express)
- **Database**: MariaDB
- **OAuth 로그인**: GitHub
